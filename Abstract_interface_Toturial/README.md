# Abstract class 與 Interface 區別
* Abstract class 可以含有:
  1.建構子(Constructor)
  2.實例變數（Instance Variable) 
  3.抽象方法（Abstract method) 
  4.需要使用extends繼承，Java只提供單一繼承功能。

* Interface 只能含有:
  1.常數(CONSTANTS)
  任何變數在Interface宣告之後都會預設被設定為final static 狀態。
  2.所有的method都不能填寫內容
  4.需要使用implement來繼承，一個class可以implement多個Interface
